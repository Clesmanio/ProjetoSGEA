Sobre o Sistema de Gestão de Eventos Acadêmicos (SGEA)

Este projeto é um exemplo desenvolvido para fins didáticos na disciplina de Programação Web do CEUB. O objetivo é demonstrar conceitos básicos de desenvolvimento web utilizando o framework Django, focando em boas práticas de backend e segurança.

Funcionalidades Chave

  • Estrutura de projeto Django (MVC)
  • Restrição de acesso por Perfil (Organizador, Aluno, Professor)
  • Sistema de Inscrição e Controle de Vagas
  • Gerenciamento de Propriedade do Evento (apenas o criador edita/deleta)
  • Simulação de Emissão e Controle de Certificados
